# Docs

## Architecture

- core：微信小游戏原生能力、环境守卫、广告管理器、跳转小程序封装，以及 Laya / Cocos 运行时检测函数
- core：微信小游戏原生能力、环境守卫、广告管理器、跳转小程序封装、抖音侧边栏能力，以及 Laya / Cocos 运行时检测函数

## Principles

- 所有环境相关判断统一收口
- 所有展示型能力 Promise 化
- 所有广告实例按类型缓存
- 引擎差异仅保留在环境检测层，不再单独拆分引擎包
- 微信原生已支持的关键能力继续向上暴露，但统一收敛到 SDK 配置和事件接口中
- 抖音侧边栏能力单独封装为通用模块，确保能在 game.js 时机同步监听 onShow 并缓存最新启动参数

## API Docs

- 面向接入方的完整方法说明与示例见根目录 README
- README 已按 init、激励视频、插屏、原生模板、跳转小程序、辅助导出 分组说明
- README 已补充抖音侧边栏接入步骤、推荐时机和最小示例
- 若后续新增能力，优先在 README 中补充“方法说明 + 最小示例”
